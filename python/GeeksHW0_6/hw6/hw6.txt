Тема: Декораторы в Python

 

📌 Задание 1 — Проверка администратора

 

Создайте класс User.

 

Атрибуты класса

 

name — имя пользователя

 

role — роль пользователя

 

Пример:

 

admin = User("Ardager", "admin")

user = User("Bek", "user")

Создайте декоратор:

 

is_admin

Этот декоратор должен:

 

• проверять роль пользователя

 

• если роль admin — функция выполняется

 

• если роль не admin — вывести сообщение:

 

У вас нет доступа

 

Создайте функцию:

delete_video(user)

Функция должна выводить:

 

Видео удалено

 

Функция должна использовать декоратор is_admin.

 

 

📌 Задание 2 — Декоратор таймера

 

Создайте декоратор:

 

timer

 

Этот декоратор должен:

 

• замерять время выполнения функции

 

• выводить сколько времени выполнялась функция.

 

Подсказка

 

Можно использовать модуль:

 

import time

Создайте функцию:

download_video()

Внутри функции сделайте задержку:

time.sleep(2)

и выведите текст:

 

Видео загружено

 

Функция должна быть обёрнута декоратором timer.

 

Пример результата

 

При запуске должно выводиться примерно так:

 

Видео загружено

Время выполнения: 2.0 секунд
