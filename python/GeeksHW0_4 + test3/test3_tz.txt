🧾 Условие задачи 
Вы — главный разработчик в фэнтези-банке Simba, который обслуживает только героев: магов, воинов и других представителей мира приключений.
Нужно создать мини-приложение, используя темы ООП: наследование, инкапсуляция, переопределение методов, магические методы, абстрактные классы.

Что нужно сделать
✅ 1. Создать базовый класс Hero
У героя должны быть:
name — имя


lvl — уровень


hp — здоровье
Метод:
action() — выводит строку:
 "{name} готов к бою!"




✅ 2. Создать два дочерних класса
MageHero — наследник Hero
Дополнительно:
mp — мана
Переопределить метод action() так, чтобы вывод был:
 "Маг {name} кастует заклинание! MP: {mp}"WarriorHero — наследник MageHero (!)
Переопределить action() так:
 "Воин {name} рубит мечом! Уровень: {lvl}"

✅ 3. Создать класс BankAccount
Этот класс демонстрирует инкапсуляцию и работу со свойствами.
У счета должны быть:
hero — объект героя
_balance — защищённый атрибут
__password — приватный атрибут
bank_name - атрибута

Методы:
login(password) — проверяет пароль
full_info — свойство (только чтение): возвращает информацию о герое и балансе
get_bank_name() — возвращает название банка
bonus_for_level() — возвращает бонус за уровень (например, lvl * 10)

✅ 4. Реализовать магические методы
__str__(self)
Возвращает строку вида:
 "{name} | Баланс: {balance} SOM"
__add__(self, other)
Складывает балансы только если герои одного класса.
 Mage + Mage — можно
 Warrior + Mage — ошибка
__eq__(self, other)
Герои считаются равными, если у них одинаковые:
класс
уровень


✅ 5. Абстрактный класс SmsService
Создать абстрактный класс с методом:
send_otp(phone) — обязательно должен быть реализован в дочерних классах
Реализовать два варианта:
KGSms
Возвращает строку:
<text>Код: 1234</text><phone>{phone}</phone>
RUSms
Возвращает словарь:
{"text": "Код: 1234", "phone": "{phone}"}
✔️ Пример использования:
mage1 = MageHero("Merlin", 80, 500, 150)
mage2 = MageHero("Merlin", 80, 500, 200)
warrior = WarriorHero("Conan", 50, 900, 20)
acc1 = BankAccount(mage1, 5000, "1234", “Simba”)
acc2 = BankAccount(mage2, 3000, "0000", “Simba”)
acc3 = BankAccount(warrior, 2500, "1111", “Simba”)

print(mage1.action())
print(warrior.action())
print(acc1)
print(acc2)
# --- Классовые и статические методы ---
print("Банк:", acc1.get_bank_name())
print("Бонус зауровень:", acc1.bonus_for_level(), "SOM")

# --- Магические методы: __add__ ---
print("\n=== Проверка __add__ ===")
print("Сумма счетов двух магов:", acc1 + acc2)

print("Сумма мага и воина:", acc1 + acc3)

# --- Магический метод: __eq__ ---
print("\n=== Проверка __eq__ ===")
print("Mage1 == Mage2 ?", acc1 == acc2)  # True — одинаковое имя и уровень
print("Mage1 == Warrior ?", acc1 == acc3)  # False

# --- SMS ---
sms = KGSms()
print("\n", sms.send_otp("+996777123456"))
