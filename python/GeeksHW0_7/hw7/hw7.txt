📌 Создать базу данных

store.db

📌 Создать таблицу

Создайте таблицу:

products

🔹 Поля таблицы

id — PRIMARY KEY

name — название товара

price — цена

quantity — количество

📌 Реализовать CRUD операции


1️⃣ CREATE — добавление товара

Функция:

create_product(name, price, quantity)

Добавляет новый товар в базу.

2️⃣ READ — получение данных


Функция:

read_products()

Выводит все товары из базы.


3️⃣ UPDATE — обновление данных


Функция:

update_product(id, price)


Обновляет цену товара по id.


4️⃣ DELETE — удаление

Функция:

delete_product(id)

Удаляет товар по id.
